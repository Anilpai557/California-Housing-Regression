# California-Housing-Regression

This project uses machine learning models to predict housing prices using the California Housing dataset.

Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

Evaluation Metrics
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
- R² Score


Random Forest performed best due to its robustness and ability to model non-linear relationships.

Tools & Libraries
- Python
- Scikit-learn
- Pandas
- NumPy
